How Backtracking Works:
1. Explore — Try each of 8 directions from current position
2. Slide — Player slides until hitting wall/boundary
3. Collect — Pick up gems passed during slide
4. Prune — Skip if path ≥ best known or state already visited with fewer moves
5. Backtrack — Undo move, try next direction

Algorithm Pseudocode
function backtrack(x, y, mask, path):
  if mask == fullMask: // All gems collected!
    update bestPath
    return SUCCESS

  if path.length ≥ bestPath.length:
    return PRUNE (too long)

  if visited[x,y,mask] ≤ path.length:
    return PRUNE (seen better)

  for each direction in 8 dirs:
    simulate slide → (newX, newY)
    collect gems along the way
    path.push(direction)
    backtrack(newX, newY, newMask, path)
    path.pop() // ← BACKTRACK!
